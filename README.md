# ArticleSpace

.NET solution for managing articles and products.

## Tech Stack

- ASP.NET Core MVC + Web API
- Entity Framework Core
- PostgreSQL
- Scalar (OpenAPI documentation)

## Architecture

- **ArticleSpace.Web** — MVC frontend
- **ArticleSpace.ApiService** — REST API with articles/products management
- **ArticleSpace.ServiceDefaults** — shared configuration
- **ArticleSpace.AppHost** — orchestration
